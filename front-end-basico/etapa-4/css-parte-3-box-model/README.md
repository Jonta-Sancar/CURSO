# CSS (parte 3) - Box Model

> Modulo: Front-end Basico

## Objetivo da aula
Entendimento de margin, border, padding e content no calculo de layout.

## Exemplo rapido
```css
.bloco {
  width: 260px;
  padding: 16px;
  border: 2px solid #1d3557;
  margin: 20px;
}
```

## Pratica sugerida (exercicio concreto)
### Projeto: Comparativo visual do Box Model

### Requisitos obrigatorios
1. Crie 3 caixas com o mesmo conteudo textual.
2. Altere apenas `padding`, `border` e `margin` entre elas.
3. Mostre uma caixa com `box-sizing: content-box` e outra com `border-box`.
4. Defina largura fixa para todas e compare o tamanho final.
5. Inclua uma legenda explicando a diferenca observada.

### Regras de validacao
- As tres caixas devem ser visualmente diferentes.
- A comparacao de `box-sizing` deve ser perceptivel.
- Nao misturar propriedades sem explicar o motivo.
- Legenda deve descrever o que mudou em cada caixa.

- Arquivo(s) HTML e CSS organizados em pasta unica da aula.
- Estrutura legivel, com nomes claros e sem conteudo placeholder.
