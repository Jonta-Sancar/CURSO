# CSS (parte 2) - Principais aplicacoes

> Modulo: Front-end Basico

## Objetivo da aula
Edicoes de texto, margens, tamanhos, proporcoes e principais tipos de seletores.

## Exemplo rapido
```css
.card {
  margin: 24px auto;
  width: 320px;
  line-height: 1.5;
}
```

## Pratica sugerida (exercicio concreto)
### Projeto: Pagina de materia com hierarquia visual

### Requisitos obrigatorios
1. Crie um bloco principal de artigo com titulo, subtitulo e 3 paragrafos.
2. Aplique estilos de fonte: tamanho, peso e altura de linha.
3. Trabalhe margens externas e internas para separar secoes.
4. Use seletores por classe, id e descendencia no mesmo arquivo.
5. Inclua uma caixa lateral de "Dica rapida" com dimensoes definidas.

### Regras de validacao
- Texto principal deve ter boa legibilidade (nao muito comprimido).
- Espacamentos devem mostrar diferenca clara entre `margin` e `padding`.
- Evitar repetir regras iguais em varios seletores.
- Nao usar unidades incoerentes no mesmo bloco sem motivo.

### Entrega esperada
- Arquivo(s) HTML e CSS organizados em pasta unica da aula.
- Estrutura legivel, com nomes claros e sem conteudo placeholder.
