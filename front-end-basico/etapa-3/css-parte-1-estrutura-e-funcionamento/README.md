# CSS (parte 1) - Estrutura e funcionamento

> Modulo: Front-end Basico

## Objetivo da aula
Como CSS funciona, sintaxe basica e relacao entre seletores e elementos HTML.

## Exemplo rapido
```css
h1 {
  color: #1f3a5f;
  font-size: 32px;
}
```

## Pratica sugerida (exercicio concreto)
### Projeto: Cartao de perfil com CSS externo

### Requisitos obrigatorios
1. Crie `index.html` e `styles.css`, ligando o CSS via `<link>`.
2. Monte um cartao com nome, cargo, bio curta e link de contato.
3. Estilize fundo, tipografia, borda e espacamentos do cartao.
4. Use ao menos 1 classe e 1 seletor por tag (ex.: `h2`).
5. Destaque visualmente o link com estado `:hover` simples.

### Regras de validacao
- Nao usar `style` inline.
- O CSS deve estar em arquivo separado.
- Seletores devem refletir a estrutura do HTML.
- A leitura do texto deve estar confortavel.

### Entrega esperada
- Arquivo(s) HTML e CSS organizados em pasta unica da aula.
- Estrutura legivel, com nomes claros e sem conteudo placeholder.
