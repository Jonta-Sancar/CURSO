# Landing Page (CTA - Captura de Lead)

> Modulo: Front-end Basico

## Objetivo
Aplicar HTML e CSS para montar uma landing page com chamada para acao (CTA).

## Pratica sugerida (exercicio concreto)
### Projeto: Landing page de inscricao para minicurso gratuito

### Requisitos obrigatorios
1. Crie as secoes: `header`, hero, beneficios, prova social, formulario e `footer`.
2. No hero, inclua titulo forte, subtitulo e 1 CTA principal visivel sem rolagem.
3. Em beneficios, apresente 3 cards com titulo e descricao curta.
4. Em prova social, inclua 2 depoimentos ou 3 metricas (ex.: alunos, aulas, avaliacao).
5. No formulario, use os campos: nome, email e checkbox de aceite.
6. No `footer`, informe empresa/remetente, email de contato e cidade.

### Regras de validacao
- CTA principal deve aparecer no topo da pagina.
- Formulario precisa ter `label` e validacao basica (`required`).
- Hierarquia de titulos deve ser coerente (`h1` unico no hero).
- A pagina deve ser responsiva em mobile (375px) e desktop (1200px).

### Entrega esperada
- `index.html` + `styles.css` com foco em conversao e leitura rapida.
- Conteudo realista (sem "lorem ipsum"), com oferta clara e objetiva.
